Goal is to visualize the injury data from previous races to better understand
the probability of accidents within the sport of motocross. Potentially strecthing out
to getting information about tracks and finding correlations to the tracks and the accidents that
occur.

- injury report websites -
https://racerxonline.com/injury-report
https://www.swapmotolive.com/page/2/?s=injury

- Documentation - 
requests - https://pypi.org/project/requests/
Selenium - https://www.selenium.dev/documentation/en/
WebDriver - 

Selenium webscraper to parse motocross web pages and determine the prevelance of certain injuries at certain motocross tracks.

Running the scripts

set a venv with the following commands

1. python3 -m venv <venv-name>
2. source <venv-name>/bin/activate
